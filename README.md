# 🔒 LockDown Total - Android Lock Screen

**Aplikasi ini mengunci HP secara total dengan:**
- Overlay penuh di atas semua aplikasi
- Getaran terus-menerus
- Senter berkedip
- Tombol fisik dinonaktifkan (Back, Home, Recent, Volume)
- Hanya bisa dibuka dengan password `123`

## ⚠️ PERINGATAN
- Gunakan hanya pada perangkat Anda sendiri
- Jika lupa password, restart HP dengan tekan power 10 detik
- Aplikasi ini membutuhkan izin Device Admin dan Overlay

## Cara Install
1. Clone repo ini
2. Buka dengan Android Studio
3. Build APK dan install
4. Aktifkan Device Admin saat diminta
5. Izinkan overlay (Draw over other apps)
